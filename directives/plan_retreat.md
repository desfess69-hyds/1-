# 수련회 기획 SOP

## 목표
교회/단체로부터 수련회 의뢰가 들어오면 **기획안 초안**을 작성한다.
초안은 서동현(사람)이 검토·수정한 뒤 클라이언트에게 전달된다.

## 입력 (사용자가 줄 정보)
- 교회/단체명
- 대상 (예: "청년부 30명", "초등부 50명")
- 날짜 / 기간 (예: "2026-08-15 ~ 17, 2박 3일")
- 예산 (예: "250만원")
- 주요 메시지/방향 (선택 — 비어있으면 AI가 제안)
- 특별 요청 (선택 — 강사 선호, 장소 선호 등)

## 사용할 도구
1. `execution/generate_retreat_plan.py` — Claude로 기획안 본문 생성
2. `data/speakers.json` — 사내 강사 풀 참조 (있으면)
3. `data/venues.json` — 장소 풀 참조 (있으면)
4. `templates/retreat_template.md` — 기획안 양식

## 절차
1. 입력 정보가 빠진 게 있으면 → 사용자에게 명확히 묻기 (지레짐작 금지)
2. `data/speakers.json`과 `data/venues.json` 읽기 → 후보 좁히기
3. `execution/generate_retreat_plan.py` 실행
   ```bash
   python execution/generate_retreat_plan.py \
     --church "○○교회" \
     --target "청년 30명" \
     --period "2026-08-15~17, 2박 3일" \
     --budget 2500000 \
     --message "회복" \
     --output ".tmp/dossiers/○○교회_2026-08_기획안.md"
   ```
4. 결과물(.tmp/dossiers/...md) 사용자에게 보여주고 수정 받기

## 출력
`.tmp/dossiers/{교회명}_{날짜}_기획안.md` 파일.

**필수 항목** (`templates/retreat_template.md`의 모든 섹션):
- 컨셉 한 줄
- 주제 말씀 (성경 본문 3안)
- 시간표 (일자별, 30분 단위)
- 강사 후보 3명 (이름·전문분야·예상 사례비)
- 장소 후보 3곳 (위치·수용 인원·예상 비용)
- 예산표 (강사료·장소·식대·교통·물품·예비비, 합계가 예산 범위 내)
- 위험 요소 (예산 초과 가능성, 강사 일정 충돌, 안전 이슈 등)

## 엣지 케이스
- **예산이 너무 적음** → 솔직히 말하고 절충안(인원 축소, 1박, 자체 강사 등) 제안
- **인원 50명 이상** → 안전 관리(소방·보험·간호인력) 항목 추가
- **강사 풀이 비어있음** → 사용자에게 "강사 데이터 없음. 외부 추천 받을지, 직접 알려주실지?" 묻기
- **종교 관련 민감 주제** → 교파(예장합동/통합, 감리교 등) 확인 필수
