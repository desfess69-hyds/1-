# 수련회 팀장 종합 체크리스트 (HYDS 마스터)

> **출처**: 서동현 기존 양식 + HYDS AI 보강 (안전·법·디지털·사후 항목 추가)
> **작성일**: 2026-05-29
> **사용처**: 신규 의뢰 받을 때 팀장에게 전달 + 점검 SOP의 기준 데이터

---

## A. 팀장 5단계 (큰 흐름)

| 단계 | 내용 | 마감 권장 |
|------|------|----------|
| **Step 1** | 교회 측과 zoom 첫 소통 — 소통 방식(카톡/문자/줌)·교회 Needs 파악 | D-90 |
| **Step 2** | HYDS 본부에 교회 Needs 전달·정리 | D-85 |
| **Step 3** | 팀원 모집·업무 배치 (예배 팀장 + 프로그램 팀장 우선) | D-75 |
| **Step 4** | 답사 (팀장 + 가능한 팀원들 + 기도회 측 교회 챔들) | D-60 |
| **Step 5** | 세부 기획 회의 — 교회와 지속적 소통 | D-45 ~ D-7 |

## B. 운영 (예배·프로그램 제외 모든 것)

### B-1. 장소·일정
- [ ] 일정 확정 (1박 2일 / 2박 3일)
- [ ] 숙소 — 스탭 숙소, 캠프 참여자 숙소 별도 확인
- [ ] 온수, 시설(화장실·샤워실 수, WiFi) 확인
- [ ] 식사 — 교회 제공 vs HYDS 도시락
- [ ] **[+] 장소 답사 시 사진·동영상 확보** (팀원 공유용)
- [ ] **[+] 비상 시 대피 동선 사전 확인** (가까운 병원·소방서 거리 메모)
- [ ] **[+] 주차 가능 대수 + 차량 진입 가능 시간**

### B-2. 캠프 (팀 구성)
- [ ] 팀원 모집 완료
- [ ] 예배 팀장 / 프로그램 팀장 세움
- [ ] **[+] 안전 담당(응급의료) 1명** 별도 지정 (간호인 우대)
- [ ] **[+] 사진·영상 담당 1명** 별도 지정

### B-3. 세부 사항
- [ ] 티셔츠 제작 여부 — 제작 시 디자인 상의 (디자인은 HYDS 제공 가능)
- [ ] 캠프 타임테이블 작성 — 폼은 요청 시 HYDS 제공
- [ ] 캠프 주제 — 교회와 상의 후 결정
- [ ] **[+] 단체 사진 시간** 시간표에 반영

## C. 예배

> 핵심: 팀장을 세우고 교회와 상의가 있는 것

### C-1. 설교자
- [ ] 교회와 상의 후 설교자 결정
- [ ] HYDS 목사님 원할 시 — HYDS 간사님과 소통
- [ ] **[+] 설교자 사례비·교통비·숙소 확정 (서면)**
- [ ] **[+] 설교 본문·메시지 방향 사전 공유 받기**

### C-2. 예배 팀장
- [ ] 인도자 세움 (인도자 = 예배 팀장이 효율적)
- [ ] 예배 큐시트 작성 (요청 시 HYDS 제공)
- [ ] **[+] 찬양곡 리스트 + 코드·악보 사전 공유**
- [ ] **[+] 반주자 일정 더블체크**

### C-3. 음향 기기
- [ ] 예배 시 필요한 음향 기기 파악
- [ ] 교회 기존 장비 외 필요 시 간사와 소통
- [ ] **[+] 마이크·인이어·케이블 여분** (예비 1세트)
- [ ] **[+] 음향 리허설 시간 시간표에 명시**

## D. 프로그램

> 핵심: 팀장을 세우면 운영이 편함

### D-1. 쉐도우 캠프
- [ ] 쉐도우 캠프 여부 결정
- [ ] 진행 시 — 프로그램 팀장 주관
  - [ ] 프로그램 설명·담당자·장소·진행방법 정리
- [ ] **[+] 프로그램별 소요시간 실측** (이전 사례 참고)

### D-2. 기획서
- [ ] 기획서 작성 (요청 시 HYDS 제공)
- [ ] **[+] 기획서 교회 측 최종 승인 받기 (서면 또는 카톡 확정)**

## E. 🆕 [HYDS 추가] 누락되기 쉬운 항목

### E-1. 안전·법·보험
- [ ] **참가자 보험 가입** (50명 이상 필수)
- [ ] **응급의료 키트** 준비
- [ ] **가까운 응급실 위치·번호** 명단
- [ ] **미성년자 보호자 동의서·연락처** 수합
- [ ] **알레르기·복용약·지병** 사전 수합
- [ ] **사고 발생 시 대응 매뉴얼** 팀장 숙지

### E-2. 교통
- [ ] **버스/차량 수배** + 견적서 보관
- [ ] **운전자 명단** + 면허 확인 + 보험 확인
- [ ] **출발 시간·집결 장소** 참가자 공지
- [ ] **귀가 시 픽업 동선** 미성년자 보호자 안내

### E-3. 참가자 관리
- [ ] **참가 신청서 양식** (이름·연락처·소속·알레르기·비상연락)
- [ ] **마감일 기준 참가자 명단 확정**
- [ ] **단톡방** 개설 + 공지 매뉴얼
- [ ] **방·조 배정표** 사전 공유
- [ ] **이름표/조끼** 미리 인쇄

### E-4. 예산·정산
- [ ] **항목별 예산표** 교회와 합의 (서면)
- [ ] **사례비·강사비·장소비 정산 일정** 명시
- [ ] **영수증 보관 담당자** 지정
- [ ] **계좌·법인카드** 사용 계획
- [ ] **사후 결산 보고서** D+7 이내 제출

### E-5. 홍보·모집
- [ ] **신청 폼** 오픈 (D-60)
- [ ] **카드뉴스/포스터** D-45 1차, D-15 마감 임박 푸쉬
- [ ] **단톡방 매일 D-day 공지**
- [ ] **모집 인원 미달 시 비상 플랜** (대상 확대 or 일정 축소)

### E-6. 기록·동의
- [ ] **사진·영상 촬영 동의서** (미성년자는 보호자 동의)
- [ ] **개인정보 수집·이용 동의서**
- [ ] **수련회 사진·영상 활용 동의** (SNS·홍보용)
- [ ] **드라이브 폴더** 사진/영상 업로드 약속

### E-7. 디지털·체크인
- [ ] **출석 체크 방식** 결정 (수기 / 단톡 / QR)
- [ ] **시간표 자동 알림** (단톡 봇 or 카톡 알림장)
- [ ] **분실물·습득물** 단톡방 채널

### E-8. 사후 (D+1 ~ D+14)
- [ ] **참가자 설문 발송** (D+3 이내)
- [ ] **결산 보고** 교회 측에 (D+7 이내)
- [ ] **강사·장소·진행 후기** HYDS 데이터에 반영
- [ ] **다음 기획에 적용할 교훈** 메모
- [ ] **참가자·교회 감사 인사**
