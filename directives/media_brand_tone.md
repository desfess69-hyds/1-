# HYDS 미디어 브랜드 톤·금기·정책 마스터 (SOP)

> **상위 문서**: `docs/CONTEXT.md` § 2.2
> **참조 의무**: 모든 미디어 본부 sub-agent (media-director, concept-planner, scriptwriter, media-producer, trend-scout)
> **갱신 책임**: 운영 학습 발생 시 `directives/learnings.md`에 한 줄, 톤 변경은 이 문서에 직접 반영.

---

## 1. 브랜드 톤 매트릭스

| 차원 | HYDS 톤 | 금기 |
|------|---------|------|
| **온도** | 따뜻하고 차분 | 차갑거나 권위적 |
| **깊이** | 1~2문장에 핵심 1개 | 추상적·교과서적 |
| **자세** | 청년 눈높이, 같이 걷는 친구 | 가르치는 형·꼰대 |
| **유머** | 가벼운 농담 OK | 비꼼·자기비하·풍자 |
| **신앙** | 본문 인용 + 적용 | 강요·교리 강조·이단 비판 |

## 2. 금기 (모든 콘텐츠 절대 금지)

- 정치 풍자·정당 비교
- 이단 비판·교파 비교 (예장합동 vs 통합 등)
- 미성년자 단독 촬영 (보호자 동의 없이)
- "꼭 보세요!", "절대 놓치지 마세요!" 같은 어그로 카피
- 신앙 강요 톤 ("이거 안 하면 죄")
- 다른 단체·교회 부정 평가
- 트렌드라도 폭력·노출·비속어 포함 밈

## 3. 콘텐츠 유형별 가이드

### 3.1 트렌드 콘텐츠 (A)
- 청년 친화 밈·챌린지만 사용
- 출처(원본 인스타·틱톡) 명시
- HYDS 적용 시 본래 의미 훼손 X

### 3.2 메시지 콘텐츠 (B)
- 성경 본문 인용 시 출처 자막 작게 (예: "마6:33")
- 1편 = 1메시지 (욕심 X)
- CTA는 "저장·공유" 외에 "오늘 한 번 시도" 같은 행동 권유

### 3.3 정보 콘텐츠 (C, 수련회 홍보 등)
- 날짜·장소·신청 링크 정확
- 마감일 명시
- 미성년자 대상이면 보호자 정보 입력 안내

## 4. 자막 디자인 표준

| 항목 | 표준 |
|------|------|
| 한글 폰트 | Pretendard Bold (또는 Apple SD Gothic Neo Bold) |
| 영문 폰트 | Inter Bold |
| 색 | 흰색 (#FFFFFF) + 검정 외곽선 2px |
| 크기 | 화면 1/12~1/8 (모바일 가독성) |
| 위치 | 화면 하단 1/3 또는 가운데 |
| 줄 수 | 최대 2줄, 한 줄 14자 권장 |
| 강조 | 핵심 단어만 노란색 (#FBBF24) 또는 굵게 |

## 5. BGM 정책

### 5.1 출처 (저작권 안전)
- **YouTube Audio Library** (무료, 상업 가능 일부)
- **Pixabay Music** (무료, 상업 가능)
- **Epidemic Sound** (유료 구독, 안전)
- **Artlist** (유료 구독, 안전)

### 5.2 금지
- 유행하는 K-pop 원곡 (저작권 위험)
- 인스타·틱톡 라이브러리 외 차용
- 출처 불명 다운로드

### 5.3 분위기 추천
| 콘텐츠 | BGM 분위기 |
|--------|-----------|
| 트렌드(밈) | 곡 자체가 밈이면 인스타 라이브러리 사용 |
| 메시지 (잔잔) | 피아노·어쿠스틱·앰비언트 |
| 메시지 (강렬) | 드라마틱 스트링·신스 |
| 수련회 홍보 | 따뜻한 인디 팝·어쿠스틱 |

## 6. 색·비주얼 톤

- 메인 컬러: 따뜻한 크림 (#FEF3E2), 다크 우드 (#4A2C1A), 앰버 (#F59E0B)
- 보조 컬러: 살구 (#FED7AA), 로즈 (#FECACA)
- 금기: 형광 네온·고채도 빨강·검정 단독 (장례식 느낌)

## 7. 검수 체크리스트 (모든 콘텐츠 발행 전)

- [ ] 위 1·2번 (톤·금기) 위반 없음

- [ ] 자막 가독성 (모바일 화면에서 확인)
- [ ] BGM 출처 안전
- [ ] 미성년자 정보 노출 없음
- [ ] 신청 링크·날짜 정확
