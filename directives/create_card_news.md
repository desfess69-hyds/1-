# 인스타 카드뉴스 제작 SOP

## 목표
주제 한 줄을 받아 인스타 카드뉴스 5~10장(1080×1080 정사각)을 만든다.

## 입력
- 주제 (예: "기도가 어려울 때 읽는 시편 3편")
- 슬라이드 수 (기본 8장, 5~10 사이)
- 톤 (기본 "따뜻하고 차분", 옵션: "도전적", "젊은톤" 등)
- 브랜드 (기본 HYDS)

## 사용할 도구
1. `execution/make_card_image.py` — Pillow로 텍스트+배경 합성
2. Claude (카피 + 캡션 + 해시태그 생성)
3. `templates/card_bg/` — 통일된 배경 이미지

## 절차
1. Claude로 슬라이드별 카피 생성:
   - 1장: 후크(궁금증 유발)
   - 2~N-1장: 본문 (1슬라이드당 1~2문장)
   - N장: CTA ("저장하세요", "공유해주세요", 팔로우 유도)
2. `execution/make_card_image.py`로 각 슬라이드 PNG 생성
3. 인스타 캡션 + 해시태그 30개 생성
4. `.tmp/card_drafts/{날짜}_{주제}/` 폴더에 저장:
   - `slide_01.png` ~ `slide_NN.png`
   - `caption.txt`

## 출력
- PNG 파일 5~10장 (1080×1080)
- 캡션 텍스트 (2200자 이내)
- 해시태그 30개

## 디자인 규칙
- **크기**: 1080×1080 정사각형 (인스타 1:1)
- **여백**: 사방 80px 안전 영역
- **폰트**: 한글은 Pretendard, 영문은 Inter (templates/fonts/)
- **컬러**: HYDS 브랜드 색 (정해진 뒤 여기 명시)
- **장수**: 5~10장 (너무 길면 이탈)

## 엣지 케이스
- 텍스트가 길어 잘림 → 자동으로 슬라이드 분할
- 성경 본문 인용 → 출처(예: "마태복음 6:33") 작은 글씨로 표기
- 민감 주제(이단·정치·교파 다툼) → Claude가 거절하고 사용자에게 보고
