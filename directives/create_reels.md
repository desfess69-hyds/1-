# 인스타 릴스 제작 SOP

> **담당**: media-producer (실행) + scriptwriter (대본) + concept-planner (컨셉) + trend-scout (선택)
> **상위 톤**: `directives/media_brand_tone.md`
> **결과물**: `.tmp/media_drafts/{YYYYMMDD}_{slug}/`

---

## 1. 30초 릴스 표준 구조 (HYDS)

| 시간 | 역할 | 글자수 | 핵심 |
|------|------|--------|------|
| 0~3초 | **후크** | 15~20자 | 공감·궁금증, 1초 내 시선 잡기 |
| 3~25초 | **본문** | 80~120자 | 메시지 1개만, 컷 5~7개 |
| 25~30초 | **CTA** | 10~15자 | 저장·공유 / 또는 행동 권유 |

### 1.1 길이 변형
- **15초**: 후크 2초 / 본문 10초 / CTA 3초 — 짧고 강한 메시지용
- **30초**: 표준
- **45~60초**: 메시지 깊이 필요할 때, 컷 더 다양하게

## 2. 컨셉 3종 (concept-planner 책임)

| 컨셉 | 언제 쓰나 | 후크 예시 |

| **공감형** | 메시지 콘텐츠 | "요즘 기도가 안 돼요" |

## 4. Vrew 친화 대본 형식 (`vrew_script.txt`)

```
요즘 기도가 안 됩니다.
입은 떨어지지 않고
마음만 무거운 날들.
[#기도 #청년 #답답함]

다윗도 그랬어요.
"내 영혼이 잠잠히
하나님만 바라는도다" (시 62:5)
[#시편 #하나님 #쉼]

오늘 1분만,
그저 침묵으로 앉아보세요.
말 없이도 들으십니다.
[#기도습관 #회복]

저장하고 하루 한 번 보세요.
@hyds.official
```

- **한 줄 = 한 컷** (Vrew 자동 분할)
- **[#키워드]** 옆에 명시 → Vrew AI 이미지 자동 매칭
- **음성 추천 댓글**: "음성: 따뜻한 여성 30대 — 미나"
- **BGM 댓글**: "BGM: 잔잔한 피아노 (Vrew 라이브러리 'Hope')"

## 5. CapCut 가이드 형식 (`capcut_guide.md`)

```markdown
# CapCut 작업 가이드 — 4분 안에 완성

## Step 1 — 템플릿 선택 (30초)
- CapCut 앱 열기 → 검색: "○○○ 챌린지"
- 추천 템플릿 ID (있으면): #12345

## Step 2 — 자료 import (30초)
- assets/ 폴더의 이미지 3장 끼워넣기

## Step 3 — 텍스트 인서트 (1분)
- 0~3초: "{후크 텍스트}"
- 3~25초 (컷별):
  - 컷1 3~7초: "{본문1}"
  - 컷2 7~13초: "{본문2}"
  ...
- 25~30초: "{CTA}"

## Step 4 — BGM (30초)
- CapCut 라이브러리 검색: "{BGM 키워드}"
- 추천 곡: "○○○"

## Step 5 — 출력 (30초)
- 1080×1920, 30fps, 인스타 권장
```

## 6. 캡션·해시태그 가이드

### 6.1 캡션 구조
1. **첫 줄 후크** (영상 후크 변주, 영상 안 본 사람도 클릭)
2. **2~4문장 본문** — 영상 메시지 부연·적용
3. **CTA** — "저장하세요 / 댓글로 알려주세요"
4. **해시태그** — 마지막 줄 또는 댓글에

### 6.2 해시태그 30개 구성
- 메인 (#기도, #말씀, #신앙) × 5
- 청년 (#기독청년, #청년부, #20대) × 5
- 콘텐츠 유형 (#릴스, #shorts, #일상릴스) × 5
- HYDS (#hyds, #하이즈) × 3
- 트렌드 키워드 × 5
- 광범위 (#기독교, #예수님, #하나님사랑) × 7
