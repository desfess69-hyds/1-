# 트렌드 정찰 SOP (trend-scout)

> **담당**: trend-scout sub-agent
> **상위 톤·금기**: `directives/media_brand_tone.md`
> **결과물 위치**: `.tmp/media_drafts/{YYYYMMDD}_{slug}/trend_brief.md`

---

## 1. 목표

기독교 청년 콘텐츠에 적용 가능한 인스타·틱톡·CapCut 트렌드를 발견·필터·번역해서, 본인이 5분 안에 CapCut에서 영상 만들 수 있는 가이드를 제공한다.

## 2. 입력

| 항목 | 필수 | 예시 |
|------|------|------|
| 주제 | ✅ | "평택교회 수련회 홍보" |
| 키워드 | (둘 중 하나) | "도착 챌린지", "Y2K", "MZ밈" |
| 자율 검색 | (선택) | trend-scout가 알아서 5개 픽 |
| 적용 플랫폼 | (선택) | 인스타 릴스 (기본) / 유튜브 쇼츠 / 둘 다 |
| 데드라인 | (선택) | "이번 주 안에" — 빨리 식는 트렌드는 제외 |

## 3. 수집 출처 우선순위

1. **사용자 제공 키워드** — 가장 신뢰
2. **`data/trend_keywords.json`** — 본인이 관리하는 관심 풀
3. **Claude의 일반 지식** (학습 데이터 컷오프 한계 — 검증 필요)
4. (Phase 2) Instagram Graph API · YouTube Data API · capcut.com 스크래핑

## 4. 필터 (3중 필터)

### 4.1 1차 — 톤·금기 필터
`directives/media_brand_tone.md` § 2 모든 금기 항목 통과해야 함. 단 하나라도 걸리면 제외.

### 4.2 2차 — 수명 라벨
| 라벨 | 정의 | 권장 |
|------|------|------|
| 🟢 성숙 | 등장 7~30일, 충분히 따라가도 OK | 추천 |
| 🟡 정점 | 등장 30~60일, 지금 안 만들면 식음 | 우선 |
| 🔴 식는 중 | 60일+ | 가능하면 제외 |
| ⚪ 영구 | 챌린지 아닌 일반 포맷 | 안전 |

### 4.3 3차 — 적합도 점수 (0~10)
- 청년 친화도 (0~3)
- HYDS 톤 부합도 (0~3)
- 우리 주제 적용 용이성 (0~2)
- 제작 시간 합리성 (0~2)

5점 미만이면 제외.

## 5. 결과물 양식 (`trend_brief.md`)

```markdown
# 이번 주 트렌드 5개 — {YYYY-MM-DD}

## 1. {트렌드 이름} 🟢 / 🟡 / 🔴
- **원본 출처**: 인스타 @○○○ ({링크}) / 틱톡 #○○ ({링크})
- **CapCut 검색어**: "○○○ 챌린지" (또는 "Y2K transition" 등 정확한 검색어)
- **수명 라벨**: 🟢 성숙 (등장 ~일)
- **적합도 점수**: 8/10
- **우리 적용 방안**:
  - 어디에: 평택교회 수련회 홍보용
  - 어떻게: "도착해보니 ○○○" 포맷에 우리 사진 끼워넣기
  - 필요 자료: 수련회 장소 사진 3장, 텍스트 인서트 1줄
- **필요 자료 리스트**:
  - [ ] 수련회 장소 사진 (가로 3장)
  - [ ] 텍스트 카피 (15자 이내)
  - [ ] BGM (CapCut 라이브러리 "○○○")
- **예상 소요**: CapCut 작업 4분
- **참고 영상**: {링크}

## 2. ... (반복)
```

## 6. 실행 흐름

1. 입력 받기 → 키워드 결정
2. `data/trend_keywords.json` 로드 (있으면)
3. Claude로 5개 후보 생성 (JSON 강제 형식)
4. 3중 필터 통과한 것만 남김 → 최소 3개, 최대 7개
5. `.tmp/media_drafts/{date}_trends/trend_brief.md` 저장
6. 본인 텔레그램에 5개 요약 발송

## 7. 자동 실행

- 매주 월요일 09:45 (`com.hyds.weekly-trends.plist`)
- 본인이 별도 명령 없이도 도착
- `data/trend_keywords.json`의 키워드를 기준으로 자율 검색

## 8. 엣지 케이스

- **모든 후보가 필터 탈락** → 솔직히 "이번 주 적합한 트렌드 0개" 보고
- **본인이 제공한 키워드가 금기 위반** → 거절 + 사유 명시
- **CapCut 검색어가 영문만 있음** → 한글 번역 표기 같이 제공
- **트렌드가 너무 빠르게 식음 (수명 < 7일)** → 노란불 경고 + 진행 여부 사용자에게

## 9. 협력

- 트렌드로 어떤 콘텐츠를 만들지 → **concept-planner**
- 대본·자막 → **scriptwriter**
- 자료 준비 → **media-producer**
- 최종 묶음 보고 → **media-director**

## 10. 학습 기록

이번 주 5개 중 본인이 실제로 만든 콘텐츠 1~2개 → 다음 주 보고서 작성 시 "지난주 결과" 짧게 언급.
실패한 트렌드(클릭 안 됨) → `directives/learnings.md`에 "○○ 트렌드는 우리 톤과 안 맞았음" 한 줄.
