# 유튜브 쇼츠 제작 SOP (릴스 → 쇼츠 변환 우선)

> **상위 SOP**: `directives/create_reels.md` (릴스 표준 먼저 적용)
> **핵심 원칙**: 릴스 1편 = 쇼츠 1편 (같은 자료, 다른 플랫폼 최적화)

---

## 1. 릴스 vs 쇼츠 — 차이점

| 항목 | 인스타 릴스 | 유튜브 쇼츠 |
|------|------------|-------------|
| 길이 | 30~90초 | 60초 이하 권장 (3분까지 가능) |
| 비율 | 9:16 (1080×1920) | 9:16 (1080×1920) — 동일 |
| 후크 | 1초 | 1~2초 (검색 기반 노출도 있음) |
| BGM | 인스타 라이브러리 + 외부 | YouTube 라이브러리 |
| 캡션 | 인스타 캡션 + 해시태그 | 제목 + 설명 + 해시태그 |
| 검색 노출 | 약함 | **강함** (검색·추천에 잘 걸림) |
| 알고리즘 | 시청 완료율 | 시청 완료율 + CTR(클릭률) |

## 2. 변환 흐름 (릴스 → 쇼츠)

1. `directives/create_reels.md`로 릴스 먼저 완성
2. 같은 자료(`.tmp/media_drafts/{slug}/`)에 쇼츠용 파일만 추가:
   - `youtube_title.txt` — 쇼츠 제목 (검색 최적화)
   - `youtube_description.md` — 설명 + 타임스탬프 + 링크
   - `youtube_tags.txt` — 태그 15~20개
3. BGM이 인스타 라이브러리 곡이면 → YouTube 라이브러리 대체 곡 추천 추가

## 3. 쇼츠 제목 양식 (검색 노출 핵심)

### 3.1 좋은 제목
- **질문형**: "기도가 안 될 때 시편 62편을 읽어보세요"
- **숫자형**: "3분 안에 회복되는 기도 한 줄"
- **궁금형**: "수련회 가면 진짜 바뀔까?"

### 3.2 나쁜 제목 (회피)
- 어그로: "충격! 이거 안 하면 큰일납니다"
- 모호: "오늘의 묵상"
- 과장: "인생 바뀝니다"

### 3.3 최적화 규칙
- 60자 이내
- 첫 30자에 핵심 키워드 (검색 노출)
- 해시태그는 제목 끝에 #shorts 1개만

## 4. 쇼츠 설명 형식 (`youtube_description.md`)

```markdown
{릴스 캡션 첫 단락 그대로}

▼ HYDS 인스타: https://instagram.com/hyds.official
▼ 수련회 신청: {링크 — 해당될 때만}

📌 이번 영상의 본문
시편 62:5 "내 영혼이 잠잠히 하나님만 바라는도다"

📌 함께 보면 좋은 영상
- {이전 시리즈 링크}

📌 HYDS 소개
HYDS는 기독교 청년·수련회를 위한 콘텐츠를 만듭니다.
구독·좋아요는 더 많은 청년에게 닿게 합니다.

#shorts #기도 #말씀 #청년 #하나님 (15~20개)
```

## 5. 쇼츠 태그 가이드

| 그룹 | 예시 | 개수 |
|------|------|------|
| 핵심 메인 | 기도, 말씀, 신앙 | 3~5 |
| 청년 | 기독청년, 청년부 | 3 |
| 콘텐츠 유형 | shorts, 쇼츠 | 2 |
| 브랜드 | hyds, 하이즈 | 2 |
| 트렌드 | (해당 영상 키워드) | 5 |
| 광범위 | 기독교, 예수님 | 3 |

## 6. 플랫폼별 비교 표 (참고)

```
같은 자료로 두 플랫폼 동시 운영 시 성과
- 인스타 릴스: 즉시 도달, 24시간 가장 활발
- 유튜브 쇼츠: 1주 ~ 1달 누적 도달, 검색 트래픽

→ 메시지·정보 콘텐츠 = 쇼츠가 장기적 가치 ↑
→ 트렌드·밈 콘텐츠 = 릴스가 즉시 효과 ↑
```

## 7. 검수 (쇼츠 발행 전)

- [ ] 인스타 릴스 검수 통과
- [ ] BGM이 YouTube에서도 안전한지 (Content ID 위험 확인)
- [ ] 제목 60자 이내·검색 키워드 포함
- [ ] 설명 첫 단락이 미리보기로 매력적인지
- [ ] 해시태그 `#shorts` 포함

## 8. 협력

- 릴스 SOP → `directives/create_reels.md`
- 자막 디자인 → `directives/media_brand_tone.md` § 4
- 자료 패키지 → **media-producer**
- 일관성 검수 → **media-director**

## 9. 학습 기록

쇼츠 발행 후 1주·1달 지표 (조회수·구독 전환·평균 시청 시간) → 다음 영상 제목·썸네일·길이 최적화에 반영.
